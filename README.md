# iridium-dashboard
### Requisites

 * Python (2.7)
 * Flask
 * Flask-SocketIO
 * Flask-MySQL
 * Gevent 
 * MySQL
 

Start the dashboard by running 

    python app.py

 
Enable the dashboard from extractor.py with the -x flag, set host or port with -y or -z flags

    extractor.py -x True -y localhost -p 5000
